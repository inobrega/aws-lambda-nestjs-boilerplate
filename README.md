# AWS Lambda NestJs Boilerplate - Domain Driven Hexagon

Boilerplate code for using NestJs with AWS Lambda following Domain Driven Hexagon best practices

### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

### Create Lambda Zip

```bash
$ npm run aws:compile
```

This command will create a zip file called "nest-lambda.zip"
